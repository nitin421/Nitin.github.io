Project 1

In this project, I imported Excel files into Power BI and created a dynamic dashboard that provides insightful visualizations on key aspects of the data profession, including: - Average Salary by Gender - Happiness with Salary - Happiness with Work-Life Balance - Favorite Programming Language - Average Salary by Job Title This dashboard not only highlights important trends in the industry but also serves as a powerful tool for making data-driven decisions.
